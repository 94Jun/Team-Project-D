## 주요 기능
**1. 회원가입 및 로그인**

![로그인](https://github.com/94Jun/img-storage/blob/main/login.png)

  - 구글 아이디를 통한 로그인 기능
  - localStorage를 사용한 자동 로그인
  - 로그인 여부에 따라 사용할 수 있는 Router 구분
